📱 Area 3: PWA Enhancements

Making the app feel more native — offline, notifications, install

Current state: Already a PWA with vite-plugin-pwa and auto-update registered. Basic offline support via service worker cache.

A

Full offline support

Work offline with todos stored in localStorage (already works for guest users). Queue changes and sync when back online. Show offline indicator banner.

Effort: Medium-High · Core infrastructure change
B

Push notifications

Due date reminders, overdue task alerts. Background sync checks for upcoming tasks even when the app is closed.

Effort: High · Requires push service, permission prompts
C

Custom install prompt

In-app "Install App" button instead of relying on the browser's default install banner. Can track dismissed/installed state.

Effort: Low · UI only, vite-plugin-pwa supports this
D

Background sync

When coming back online, automatically sync any changes made offline (todo additions, edits, completions). No data loss.

Effort: Medium · Needs service worker + sync manager
E

Better splash screen & icons

Custom branded splash screen on app launch. Higher-res icons for modern devices. Adaptive icons for Android.

Effort: Low · Asset generation
F

App badge for overdue tasks

Show a numbered badge on the app icon indicating how many tasks are overdue. Clears when tasks are completed.

Effort: Low-Medium · Uses Badge API
