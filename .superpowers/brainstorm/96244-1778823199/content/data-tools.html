🔧 Area 6: Data Tools

Power features for managing, moving, and organizing task data

A

Import / Export

Export all todos as JSON or CSV. Import from JSON/CSV (with dedup by title or ID). Useful for backup or migrating between accounts.

Effort: Medium · Parse logic, file handling, merge strategy
B

Bulk edit

Select multiple tasks → edit tags, priority, category, or due date in one action. A batch edit panel replaces the per-item edit modal.

Effort: Medium · New UI + store methods
C

Recurring tasks

Set tasks to repeat daily, weekly, monthly, or on a custom schedule. Auto-creates the next occurrence when completed. Already partially implemented per the codebase.

Effort: Medium · Scheduling logic, UI for recurrence rules
D

Smart lists / saved views

Save filter combinations as named views that appear in the sidebar/dropdown. "Overdue & Important", "Weekend Chores", "Work This Week" — one click to apply.

Effort: Medium-High · Persist views, manage in settings
E

Priority bulk actions

Batch archive completed tasks (one-click cleanup). Batch mark as high priority. "Mark all done" for visible tasks.

Effort: Low · Reuses existing batch operations
