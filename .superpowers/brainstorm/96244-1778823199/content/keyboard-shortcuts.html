⌨️ Area 2: Keyboard Shortcuts

Global and context-aware hotkeys to speed up common actions

Current state: Ctrl+N / Cmd+N opens the add-todo form. That's it. Let's expand.

A

Quick-add from anywhere

Q — Opens a compact quick-add dialog from any page. Title + optional fields without navigating to /tasks.

B

Focus search bar

/ or Ctrl+F — Focus the search/filter bar from anywhere on the page.

C

Navigation shortcuts

G then T → /tasks · G then B → /board · G then C → /calendar · G then S → /stats · G then A → /archived

D

Todo list actions

Space → toggle complete · E → edit selected · Delete → delete with confirmation · A → archive

E

Batch selection shortcuts

Ctrl+A → select all visible · Ctrl+Shift+A → deselect all · Shift+Space → select range (already works on some pages)

F

Shortcut cheat sheet

? — Toggles a modal overlay showing all available shortcuts, grouped by category.

G

Undo / Redo

Ctrl+Z → undo last action · Ctrl+Shift+Z → redo. Works for move, archive, delete, complete.
